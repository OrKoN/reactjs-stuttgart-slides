When a new project starts:

 

  1. decide what is state and what is property
  2. decide what is routable and what is not
  3. plan project structure carefully
  4. plan more time for evaluating and selecting libraries
  5. learn newest approaches in writing react apps (flux, redux etc.)

Conclusion

  • package manager that integrates with System JS
  • works with GitHub and NPM
  • loads any format
  • integrates with Babel or Traceur out-of-the-box
  • bundles/minifies out-of-the-box
